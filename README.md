# 🏦 SimpleBank dApp

A modern **decentralized banking application (dApp)** built using **Next.js, Tailwind CSS v4, Zustand, and Solidity**, deployed on the **Sepolia Ethereum testnet**.

---

## 🎨 Frontend Preview

![SimpleBank UI](https://i.ibb.co/wZqLn7RQ/n-Gz-QL2-Gt-TJWe9lvl-Bj1yug.png)

---

## 🚀 Features

- 🔐 Connect wallet via MetaMask  
- 💰 Deposit ETH into smart contract  
- 💸 Withdraw ETH securely  
- 📊 Real-time balance updates  
- ⚡ Fast and responsive UI (Next.js + Tailwind v4)  
- 🧠 State management with Zustand  

---

## 🛠 Tech Stack

- **Frontend:** Next.js (App Router) + TypeScript  
- **Styling:** Tailwind CSS v4  
- **State Management:** Zustand  
- **Blockchain:** Solidity + Hardhat  
- **Library:** Ethers.js v6  
- **Wallet:** MetaMask  
- **Network:** Sepolia Testnet
